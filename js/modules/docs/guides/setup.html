安装dust

通过npm安装

npm install --save --production dustjs-linkedin
npm install --save --production dustjs-helpers

Dust需要Node 0.8.12以上的版本

编译命令dustc在./node_modules/.bin/dustc.如果要在全局使用dustc命令,请将dustjs通过 npm install -g 安装为全局.

通过Bower安装

bower install dustjs-linkedin

Dust兼容于： IE7+ ， 各种现代浏览器.

下载

可以从这里 dustjs的releases下载到最新版本和相关的历史版本

加载Dust

Node

var dust = require('dustjs-linkedin');

Browser

在页面中引用 dust-core.min.js or dust-full.min.js文件.

如果在浏览器中编译模板，请引用 dust-full.min.js.（注：如果dust模板已经过预编译处理，只需要引用dust-core.min.js）

AMD

自Dust 2.6.0起, 你可以用AMD适配的加载方式引用，比如使用 require.js

为了保持向后兼容性,你必须通过config配置，告诉Dust将自身注册为AMD模块。 (jQuery用了同样的方式注册为AMD模块).

define.amd.dust = true;

在完成配置后，你可以用AMD模块的方式加载 dust-core.js or dust-full.js.

<script src="r.js"></script>
<script type="text/javascript">
    define.amd.dust = true;
    require(["lib/dust-full"], function(dust) {
        dust.render(...);
    });
</script>

Helpers

如果你使用dustjs-helpers 1.6.0以上的版本, dust-helpers 也被注册为AMD模块. 如果在Dust已经加载后，require了Helper模块，即可很好的使用了。 

define.amd.dust = true;
require(["lib/dust-full"], function(dust) {
    require(["lib/dust-helpers"], function() {
        // dust helpers are available when you call dust.render()
    });
});
